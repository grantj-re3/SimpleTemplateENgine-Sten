Demo of Sten features

Sten 'file_tpl' feature

The whole main pane on this page is produced via a file-template. That is, the common html-template is replacing a token with a (child) html-template file (which can also contain replacement tokens).

Sten 'file_glob' feature

A Sten 'file_glob' token is producing the whole side-pane. That is, the token will be replaced with the contents of all files which match the prefix given by the token-value and match the suffix ".inc.html"

Sten 'last_mod' feature

A Sten 'last_mod' token is producing the date/time seen at the bottom of this page after the 'Last modified' text.

The table below is produced via the 'file_csv' token. (This feature is currently experimental.)
